# Re Reference AI (RRA)

**URL**: https://rra.reload.co.jp

**Tagline**: The AI Technical Reference

## Overview

Re Reference AI (RRA) is a technical reference site for AI, LLMs, AI Agents, Machine Learning, and Generative AI.

Unlike a glossary, RRA aggregates:

- Definitions
- Official specifications
- GitHub repositories
- Documentation
- Research papers
- RFCs
- SDKs
- Version history
- Adoption examples
- Technical comparisons

## Goals

- Become the primary technical reference for AI engineers.
- Provide links to primary sources.
- Explain history and background.
- Offer implementation examples.
- Cross-link related technologies.
